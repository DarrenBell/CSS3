Fun with CSS3

Border Radius

The border-radius propety is used to create rounded corners. It has 4 values which define each corner, top left, top right, bottom right and bottom left. A good resource on border-radius can be found over at css info.

Box Shadow

The box-shadow property is used to create a drop shadow effect. It has 5 values - horizontal length, vertical length, blur radius, spread radius and finally the color. You can also set the opacity using rgba (0,0,0,X) where the X is the value for opacity between 0 and 1. You can also inset the box shadow using the inset value followed by the other values. A more in depth article can be found here. This gives the box a 3d effect.

CSS3 Backgrounds

CSS3 has new background properties, such as background-size and background-origin. Background-size specifies the size of a background image. This size can be specified in percent or pixels. The background-origin property specifies the positioning area of the background images. The background image can be placed within the content-box, padding-box or border-box area.

CSS3 also allows for multiple background images to be used. background: url(image1.png), url(image2.png);
